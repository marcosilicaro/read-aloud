Firebase Setup

Firebase is a free Google service that stores your data and handles sign-in. You'll create your own project (5 min, free forever for personal use), so your data stays in your own account.

1
Create a Firebase project
Go to console.firebase.google.com → Add project. Give it any name (e.g. "read-aloud"). You can skip Google Analytics.
2
Enable Google Sign-In
In your project → Build → Authentication → Get started → Sign-in method → Google → Enable. Pick a support email and save.
3
Create Firestore database
Build → Firestore Database → Create database → Start in production mode → pick a location. Then go to the Rules tab and paste this (only lets each user read/write their own data):
rules_version = '2'; service cloud.firestore { match /databases/{db}/documents { match /users/{uid} { allow read, write: if request.auth != null && request.auth.uid == uid; } } } Click Publish.
4
Register a Web app
Project Settings (gear icon top-left) → scroll to Your apps → click the </> (Web) icon → give it any nickname → Register app. You'll see a firebaseConfig object. Copy it.
5
Add this app's domain to authorized domains
In Authentication → Settings → Authorized domains, add any domain you'll use the app from (e.g. localhost is already there; add 192.168.1.172 or your Cloudflare Pages domain if you deploy).
6
Paste the firebaseConfig below
Paste the entire firebaseConfig = {...} object here. The app parses it.
